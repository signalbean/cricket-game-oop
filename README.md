# Cricket Game

Console-based cricket mini-game in Python. Choose a shot (1-6), if the bowler's random delivery matches your shot, you're out!

## Run

```bash
python main.py
```

## Rules

- Enter 1-6 as your shot
- Match = OUT, otherwise score increases
- Game ends when you're out

## Structure

```
main.py      # Entry point with menu & high score tracking
game.py      # Game logic, Player, Bowler, colors
```
